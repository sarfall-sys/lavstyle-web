# 🛒 Product Catalog Web Application

A full-stack product catalog application built with **Laravel** and **React**.  
The project provides an interactive and user-friendly way to browse, search, filter, and compare products.

---

## 📌 Overview

This application allows users to explore a product catalog with advanced features such as pagination, filters, search, and product comparison.  
It also includes quality-of-life features like recently viewed products and an FAQ chatbot to assist users.

## ✨ Features

- 📋 Product catalog with category menu
- 🔍 Product search
- 🧰 Advanced filters
- 📄 Pagination
- ⚖️ Product comparison
- 👀 Recently viewed products
- 🤖 FAQ chatbot for user assistance

---

## 🧱 Tech Stack

### Backend
- Laravel
- REST API
- MySQL

### Frontend
- React
- Vite
- Axios
- React Router
